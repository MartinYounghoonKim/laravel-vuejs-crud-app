# Laravel & Vue JS CRUD App

## Build Setup

``` bash
# 1. Install all laravel dependencies
composer install

# 2. Copy .env file
cp .env.example .env

# 3. Generate application key
php artisan key:generate

# 4. Serve the app
php artisan serve


# 1. install dependencies
npm install

# 2. build for production with minification
npm run dev
```
